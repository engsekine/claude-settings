現在のブランチの変更差分を確認し、変更されたページ・コンポーネントのスクリーンショットを保存します。Playwright を使用します。

## 使い方

```
/screenshots
```

## 手順

### 1. 変更ファイルの特定

```bash
git diff --name-only main...HEAD
```

変更ファイルが存在しない場合は「変更差分がありません」と出力して終了。

### 2. 影響ページの特定

変更ファイルの種類に応じて、スクリーンショットを撮影すべきURLを特定する。

- `src/pages/` または `app/` 配下: 対応するルートURL
- `src/components/` 配下: そのコンポーネントが使われているページ
- グローバルCSS・共通コンポーネント: すべてのページが対象

ローカル開発サーバーが起動していない場合は「開発サーバーを起動してから実行してください（例: npm run dev）」と出力して終了。

### 3. 保存先フォルダの作成

```bash
DATE=$(date +%Y%m%d)
NEXT=$(ls screenshots/$DATE/ 2>/dev/null | grep -E '^[0-9]+$' | sort -n | tail -1)
NEXT=$((${NEXT:-0} + 1))
mkdir -p screenshots/$DATE/$NEXT
```

### 4. スクリーンショットの撮影

Playwright を使い、各URLを以下の3幅で撮影する:

| 幅 | 高さ | ファイル名 |
|----|------|-----------|
| 1920px | 900px | `<ページ名>-1920.png` |
| 1100px | 900px | `<ページ名>-1100.png` |
| 375px | 900px | `<ページ名>-375.png` |

撮影方法:
Playwright のコマンドラインまたはスクリプトを使用してスクリーンショットを撮影する。

### 5. 完了報告

```
📸 スクリーンショット保存完了
═══════════════════════════
保存先: screenshots/<YYYYMMDD>/<N>/
撮影ページ: <ページ数>ページ
撮影ファイル: <ファイル数>ファイル

保存ファイル:
- screenshots/<YYYYMMDD>/<N>/<ページ名>-1920.png
- screenshots/<YYYYMMDD>/<N>/<ページ名>-1100.png
- screenshots/<YYYYMMDD>/<N>/<ページ名>-375.png
...
```
