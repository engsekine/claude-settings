# コーディングスキル集

修正指示から学んだパターンを蓄積し、コーディング品質の向上に活用します。
`/code-fix` コマンド実行時に自動的に追記されます。

---

## 記録フォーマット

```
### YYYY-MM-DD: <修正タイトル>

**問題**: <どんな問題があったか>
**修正**: <どのように修正したか>
**対象ファイル**: <ファイルパス>
**カテゴリ**: React / TypeScript / HTML / CSS / PHP / 汎用

Before:
\`\`\`
<修正前のコード>
\`\`\`

After:
\`\`\`
<修正後のコード>
\`\`\`
```
